# Dhanush Kandhan | Portfolio

A minimalist, high-performance portfolio built with **Astro**, **Three.js**, and **Tailwind CSS**. Designed for visibility, craft, and speed.

## ✨ Features

- **Multilingual Greeting**: A dynamic "Green Decrypt" typography animation that welcomes visitors in 9+ languages.
- **Three.js Background**: A sitewide, interactive wave animation that persists across navigations.
- **Command-Based Navigation**: Quick navigation via keyboard shortcuts (press `man` for help).
- **Content-First Design**: Easy-to-manage sections for Projects, Books, Talks, and Writing powered by JSON.
- **Clean Aesthetic**: A "neat and tidy" UI with a strong focus on typography and dark mode.

## 🚀 Tech Stack

- **Framework**: [Astro 5](https://astro.build/)
- **Animation**: [Three.js](https://threejs.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Runtime**: [Bun](https://bun.sh/)
- **Typography**: Inter / Mono fonts

## 🛠️ Getting Started

### Prerequisites

- [Bun](https://bun.sh/) installed on your system.

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd dhanu-portflio
   ```

2. Install dependencies:
   ```bash
   bun install
   ```

### Development

Start the local development server:
```bash
bun dev
```

### Build

Create an optimized production build:
```bash
bun run build
```

## 📂 Content Management

Most of the site content is stored in `src/content/`:
- `projects.json`: Curated works and tools.
- `books.json`: Reading list and reviews.
- `home-data.json`: Homepage bio and interests.
- `tools.json`: Hardware and software setup (Uses page).
